# cpp_milestone_4

Comprehensive C++ learning modules covering object-oriented programming fundamentals through advanced polymorphic design.

## Overview

**cpp_milestone_4** contains C++ learning modules 00 through 04, progressively building from basic syntax to object-oriented design patterns.

## Modules

### Module 00: Namespaces, Classes, Member Functions
- Introduction to C++ syntax
- Namespace declaration and usage
- Class definition and member functions
- Input/output streams
- String manipulation

### Module 01: Memory Allocation, References, Pointers
- Memory allocation with `new` and `delete`
- References vs pointers
- Filestream operations
- String to numeric conversion
- Object lifetime management

### Module 02: Ad-hoc Polymorphism, Operator Overloading, Fixed-Point Numbers
- Operator overloading
- Function overloading
- Fixed-point arithmetic implementation
- Stream operators
- Comparison operators

### Module 03: Inheritance
- Class inheritance
- Virtual functions
- Constructor/Destructor in inheritance hierarchies
- Method overriding
- Multiple inheritance concepts

### Module 04: Polymorphism, Abstract Classes, Interfaces
- Abstract base classes
- Virtual functions and dynamic dispatch
- Interface implementation
- Virtual destructors
- Design patterns introduction

## Key Concepts Covered

- Object-oriented programming (OOP)
- Encapsulation and access specifiers
- Inheritance hierarchies
- Virtual functions and polymorphism
- Operator overloading
- Resource management (RAII)
- Modern C++ best practices
- Memory safety and management

## Compilation

```bash
cd cpp_00
make
./program
```

Each module has its own Makefile and compilation instructions.

## 42 School Standards

All code follows strict 42 School coding standards:
- Orthodox Canonical Form for classes
- No standard containers (initially)
- No STL usage restrictions vary by module
- Specific function naming conventions
- Proper const correctness

## Learning Path

Progress through modules sequentially:
1. **cpp_00**: Syntax fundamentals
2. **cpp_01**: Memory management
3. **cpp_02**: Operator overloading
4. **cpp_03**: Inheritance
5. **cpp_04**: Polymorphism & design patterns
